# Showcase Starter Projects

These folders are starter scaffolds for portfolio projects. They are intentionally separate from the main Vite portfolio so the portfolio can still deploy as a simple static site.

## Projects

- `mern-blog-cms`: MongoDB + Express + React blog CMS starter.
- `rest-api-swagger`: Express REST API with OpenAPI/Swagger documentation.
- `ai-powered-app`: React + Express AI writing assistant skeleton.

Each project has its own `package.json` and can later be moved into a separate GitHub repo.
